Program Akademik

SMA Modern menawarkan berbagai program akademik unggulan untuk mendukung pembelajaran siswa. Berikut adalah rincian program akademik kami:

No Program Deskripsi
1 Program IPA Program studi yang fokus pada mata pelajaran Fisika, Kimia, dan Biologi untuk siswa yang ingin melanjutkan ke bidang sains dan teknologi.
2 Program IPS Program studi yang fokus pada mata pelajaran Ekonomi, Geografi, dan Sosiologi untuk siswa yang ingin melanjutkan ke bidang sosial dan humaniora.
3 Program Bahasa Program studi yang fokus pada penguasaan bahasa asing seperti Bahasa Inggris, Bahasa Jepang, dan Bahasa Mandarin.
4 Ekstrakurikuler Berbagai kegiatan tambahan seperti seni, olahraga, dan sains untuk mendukung pengembangan minat dan bakat siswa.
